메인캐릭터
- 전투, 마을에서 사용하는 플레이어가 직접 조작하는 캐릭터
- PlayerManager에서 한번 생성하면 DontDestoryOnLoad 로 관리함.
- SpriteLibraryBuilder로 외형 커스터마이징 가능
- 장착한 장비에 연결된 Ability를 가져와서 사용함
- 적을 잡아서 얻은 경험치로 레벨업 => 탑의 힘 선택
- 탑의 힘 ex) 파이어볼, 환영 단검, 무기 강화 등

메인캐릭터의 스탯
- 이동속도 => 캐릭터의 이동과, 구르기 거리에 관여.
- 공격력 => 무기 및 탑의 힘 등 모든 공격력에 기반이 되는 스탯.
- 방어력 => 해당 값 만큼 최종 대미지에서 차감

메인캐릭터의 장비
- 무기 => 공격력에 관여 + 전투에서 사용하는 기본 Ability (칼질, 활질 등)
- 방패 => 방어력에 관여 + 가드 사용 가능
- 헬멧,갑옷 등 => 방어력에 관여


메인캐릭터의 액션
- 구르기 => 앞으로 빠르게 구른다.
- 가드 => 방패 착용시에만. 전방의 공격을 일정시간동안 감소시킨다.
- 퍼펙트 가드 => 피해를 완전히 막는다.

서브캐릭터는 고유 액티브(주로 버프), 고유 패시브를 가진다.
- 인게임에서 메인 => 서브로 교체가 가능함.
- 교체는 일시적이며, 캐릭터마다 유지시간이 다르다.
- 교체는 글로벌 쿨타임이 존재한다.
- 스탯은 메인캐릭터의 레벨을 따라간다.
